# Anker Chicken

Official website for Anker Chicken.

## About Anker Chicken

Anker Chicken is a leading food business committed to quality, freshness, and great taste.

## Technologies Used

- React 19.1.1
- TypeScript
- Vite
- Tailwind CSS

## Run Locally

**Prerequisites:** Node.js

1. Clone this repository:
   ```
   git clone https://github.com/KGB69/anker-chicken.git
   cd anker-chicken
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Run with a single command (starts frontend + backend):
   ```
   npm run dev:full
   ```

4. Open http://localhost:5173 in your browser

## Deployment

This site is configured for deployment on Render.
